# Finish Taxonomy

Use this reference when choosing or describing print finishes.

## Common Finishes

| Finish | Real visual behavior | Avoid |
|---|---|---|
| Spot UV / 局部UV | Clear glossy varnish over existing print; reveals itself through angle-dependent highlights and slightly deeper contrast | White paint, wet glue drops, full-area glare, changing text color |
| Raised UV / 厚UV | Clear raised varnish with rounded micro-edge highlights; best on logos, patterns, icons | Excessive height, jelly-like blobs, imprecise bleed |
| Hot foil / 烫金烫银 | Metallic directional reflection, crisp edges, slight pressed impression | Pure yellow fill, glowing neon, noisy glitter everywhere |
| Emboss / 击凸 | Shallow raised surface, edge highlight and opposite shadow, follows substrate | Detached 3D letters, thick plastic sticker |
| Deboss / 压凹 | Subtle recessed impression and soft inner shadow | Black outlines, holes, exaggerated engraving |
| Matte varnish / 哑油 | Low sheen, clean diffuse surface | Dirty grey haze |
| Gloss varnish / 亮油 | Broader glossy reflection than spot UV if applied over an area | Mirror-like plastic unless specified |
| Soft-touch / 触感膜 | Velvety low-gloss smoothness | Furry texture, rubber look |
| Texture varnish / 纹理油 | Fine tactile pattern visible under grazing light | Random dirt/noise |
| Holographic / 镭射 | Angle-dependent rainbow shift | Full rainbow wash unless requested |

## Intensity Language

- **Subtle**: premium, realistic, mostly visible at highlight angles.
- **Medium**: visible in normal product photography, still believable.
- **Strong**: showpiece render; use only when the user asks for obvious process display.

## Quality Checks

- Finish must sit on the same perspective plane or curved surface as the packaging.
- Finish should not cover small text unless specifically requested.
- Gloss direction should match the scene light and object curvature.
- Metallic effects should have dark and bright zones, not uniform flat color.
