# Output Format

Use this structure after the user has chosen or accepted a material/process recommendation.

## 1. 图片识别结果

```text
图片类型：
推测盒型/包装类型：
结构判断：
主要展示面：
需要注意的位置：
不确定项：
```

Mention uncertainty instead of pretending the structure is confirmed when dieline quality is unclear.

## 2. 推荐材质方案

```text
主材质：
表面处理：
印刷方式：
适合原因：
成本判断：
风险提醒：
```

## 3. 工艺分布方案

| 区域 | 推荐工艺 | 目的 | 注意事项 |
|---|---|---|---|
| Logo | 烫银 / UV / 击凸 | 强化品牌识别 | 不要离折线太近 |
| 产品名 | 局部 UV | 提高层级 | 字号不能太小 |
| 背景纹理 | 逆向 UV / 压纹 | 增加高级细节 | 控制面积 |
| 边框线 | 专色 / 烫金 | 增强结构感 | 避免过细 |

## 4. 效果图生成提示词

Include:
- 包装类型/盒型。
- 材质。
- 表面处理。
- 区域工艺。
- 光线、角度、背景、画面风格。
- 是否展示刀模结构或成品包装。
- 是否留白。
- Text protection and “禁止生成乱码文字”.

Template:

```text
基于用户上传的包装图生成一张干净清爽的包装工艺效果图（设计作品集品质）。包装类型为「{包装类型}」，材质为「{材质}」，表面处理为「{表面处理}」。整体风格为「{风格方向}」。

包装主体以「{展示角度}」展示，保留清晰的结构逻辑、主要展示面、原始 logo、中文文字、英文文字、图形、比例和版式。Logo 区域使用「{Logo工艺}」，产品名称区域使用「{产品名工艺}」，主视觉/背景区域使用「{背景或主视觉工艺}」。工艺表现真实、克制、有印刷质感，不要做成夸张金属贴图。

画面为设计作品集级别（Behance/Dribbble 精选作品质感）。纯净白色或极浅灰白背景，大面积负空间留白。明亮通透的影棚柔光（大型柔光箱+白色反光板），干净柔和的浅灰地面投影，阴影通透不脏。包装色彩纯正准确，无任何色偏或灰雾罩感。包装边缘清晰锐利，材质厚度真实，折角/曲面自然。工艺反光符合真实印刷逻辑，可以体现局部工艺细节（烫金边缘、局部 UV 反光、击凸阴影、压纹肌理）。

保留原始包装上的所有中文文字、logo、版式和图形，不要重新生成文字，不要篡改字形，不要出现乱码、错字或变形中文。仅改变指定区域的材质反射、烫金、UV、击凸或压凹效果。
```

## 5. 负面提示词

Always include:

```text
不要乱码文字，不要错误品牌名，不要错误 Logo，不要多余文字，不要廉价塑料感，不要淘宝风，不要杂乱背景，不要过度反光，不要夸张金属渐变，不要变形盒体，不要错误折线，不要错误糊口，不要多余盒盖，不要结构混乱，不要水印，不要低清晰度。

【画面清洁度专项禁止】：
不要灰蒙蒙的画面色调（像蒙了灰纱布），不要脏污阴影（发黑/发黄/有噪点/边缘毛糙），不要色彩偏移（偏暖黄/偏冷蓝/偏灰绿），不要过度 HDR 处理（暗部死黑/高光溢出），不要噪点或颗粒感（除非特种纸纹理），不要背景脏污或不均匀（暗角/渐变污渍/杂色块），不要白色区域发灰发黄（白色必须是纯正白色），不要饱和度异常（过度鲜艳或灰暗褪色），不要暗部细节糊成一团（阴影必须通透有层次），不要廉价样机感，不要假中文，不要随机字母。
```

For annotated image workflows, also include:

```text
不要生成红框，不要生成标注线，不要生成箭头，不要把工艺标注画进最终效果图。
```

## 6. 印前注意事项

Always remind:
- 效果图只能用于设计提案和视觉确认，不能替代正式印刷文件。
- 打样前需要确认刀模尺寸、出血、安全边距、折线、裁切线和糊口位置。
- 出血一般建议 3mm，但最终以印厂要求为准。
- 文字建议转曲；小号黑字建议使用单黑。
- 烫金、局部 UV、击凸、压凹、压纹、白墨、专色等需要单独工艺图层。
- 白墨需单独图层，尤其透明 PET、银龙 PET、金银卡、透明窗口材料。
- 刀模线不能混入印刷图层。
- 折线和裁切线需要区分。
- 工艺区域避免压线、折线、糊口和裁切边。

## 7. 可交付文件建议

Recommend only when useful:
- 原始印刷文件：AI/PDF，文字转曲。
- 刀模线图层：DIELINE / CUT / CREASE 独立专色图层。
- 工艺图层：FOIL / SPOT_UV / EMBOSS / DEBOSS / WHITE_INK / VARNISH。
- 效果图：PNG/JPG 2K or higher for proposal.
- 工艺说明表：区域、工艺、颜色/专色、注意事项。
