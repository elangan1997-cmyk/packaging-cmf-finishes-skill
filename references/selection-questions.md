# Selection Questions

Use these when the user has not specified material, surface treatment, finish, finish region, or render style. Keep questions concise; do not dump every section if the user only needs one decision.

## A. Packaging Positioning

请选择包装定位：

A. 普通电商款，控制成本  
B. 中端品质款，质感比普通包装更好  
C. 高端品牌款，需要明显工艺感  
D. 礼盒款，需要更强仪式感  
E. 极简高级款，少工艺但质感要好  
F. 环保自然款，强调纸感和克制设计  

## B. Main Material

请选择主要材质：

A. 白卡纸，适合常规彩盒、插舌盒、扣底盒  
B. 单铜纸，适合色彩印刷和中端包装  
C. 灰板裱纸，适合天地盖、抽屉盒、礼盒  
D. 牛皮纸，适合环保、自然、手作风格  
E. 特种纸，适合高级品牌、香氛、茶叶、礼品类  
F. 金银卡纸，适合高亮、金属质感、节庆类包装  
G. PET / PVC 透明材质，适合透明盒、窗口盒、展示包装  
H. 瓦楞纸 / 坑纸，适合运输、电商、外箱、飞机盒  
I. 不干胶材质，适合瓶贴、标签、封口贴  
J. 不确定，希望根据产品推荐  

## C. Surface Treatment

## G. Bottle Material（瓶器材质，瓶装专用）

请选择瓶身材质：

A. 玻璃（高白料/晶白料/普白料），适合饮品、调味品、化妆品、白酒  
B. PET（聚酯透明瓶），适合饮料、日化、洗发水、沐浴露  
C. PP / PE / HDPE（软硬塑料瓶），适合药瓶、日化、食品、洗护  
D. 亚克力 / PMMA（厚壁高透），适合高端护肤品罐、香水瓶  
E. 铝 / 马口铁（金属瓶），适合喷雾、香水、功能饮料  
F. 陶瓷，适合高端白酒、工艺品、香薰  
G. 不确定，希望根据产品品类推荐  

## H. Bottle Body Finish（瓶身表面工艺，瓶装专用）

F. 电镀 / 真空镀（镜面金属光泽，金银铬色）  
G. 蒙砂 / 磨砂（玻璃专用，半透雾面）  
H. 丝印（油墨直接印在瓶身曲面）  
I. 烫金/烫银（曲面热烫金属箔）  
J. 烤花 / 贴花（玻璃/陶瓷专用，高温烧结）  
K. 激光雕刻（精细蚀刻文字/图案）  
L. 高光/亚光注塑（模具直接成型表面，PP/PE/PET专用）  
M. 不确定，希望根据瓶型和品类推荐  

## I. Label Application Type（标签方式，瓶装专用）

请选择标签如何贴在瓶身上：

A. 不干胶标签（最常见，纸类或膜类压敏胶直接粘贴）  
B. 收缩套标（PETG/PVC 热收缩膜 360°包裹瓶身，适合异型瓶）  
C. 模内标签 / IML（注塑时标签嵌入瓶身，一体成型，仅PP/PE瓶）  
D. 直接丝印（油墨直接印瓶身，无标签触感，极简高端）  
E. 热转印（图案从膜转移到瓶身，适合精细多色图案）  
F. 吊牌+贴标组合（高端酒类/礼品常用）  
G. 不确定，希望根据瓶型和品类推荐

## Minimal Clarification Template

When the user only says “做高级一点”:

```text
我可以先帮你做推荐方案。为了避免工艺乱加，先确认 5 个点：
1. 包装定位：普通零售 / 中高端 / 礼盒 / 极简高级 / 环保自然？
2. 材质方向：白卡 / 灰板裱纸 / 特种纸 / PET不干胶 / 牛皮纸？
3. Logo 是否需要烫金、烫银、UV 或击凸？
4. 产品名是否需要重点工艺？
5. 效果图要白底展示，还是高级品牌氛围图？
如果你不确定，我可以按“中高端品牌款”推荐一套稳妥方案。
```
