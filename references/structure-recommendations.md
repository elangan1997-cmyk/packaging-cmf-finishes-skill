# Structure Recommendations

Use this reference when recommending materials and finishes by packaging structure.

## 天地盖 / 抽屉盒 / 礼盒

Recommend:
- 灰板裱特种纸、铜版纸裱灰板、艺术纸裱灰板。
- 哑膜、触感膜、不覆膜纸感方案。
- Logo 烫金/烫银/击凸；产品名局部 UV；背景轻压纹。
- Suitable for high-end gifts, brand sets, cosmetics, tea, wine, supplements, boutique products.

Avoid:
- Overfilled foil/UV.
- Large glossy lamination if the brand wants premium subtlety.
- Processes too close to folds, lid edges, or box corners.

## 插舌盒 / 扣底盒 / 常规彩盒

Recommend:
- 白卡纸、单铜纸、白卡覆哑膜/亮膜。
- Logo 局部 UV; small foil accents for premium lines; feature badge spot UV.
- Cost-aware retail/e-commerce finish plans.

Avoid:
- Heavy gift-box processes that mismatch low-cost structures.
- Thick UV or emboss over crease/tuck areas.
- Dense small text with foil/UV.

## 飞机盒 / 瓦楞盒 / 电商运输盒

Recommend:
- 瓦楞纸、坑纸、牛皮纸、白卡裱坑。
- Single-color or spot-color printing; bold brand marks; simple line graphics.
- Minimal finish; use structure and material texture as the premium point.

Avoid:
- Fine gradients and tiny copy on rough corrugated material.
- Complex foil/UV over compression folds.
- Making a shipping box look like a rigid luxury gift box unless requested.

## 瓶贴 / 标签 / 套标

Recommend:
- 银龙 PET、哑银 PET、白 PET、透明 PET、铜版纸不干胶。
- Spot UV, raised UV, white ink underprint, foil, reverse UV, tactile varnish.
- For transparent PET, mention white ink underprint for readability.
- For metallic PET, control contrast; do not make the whole label mirror chrome unless requested.

Avoid:
- Too-thin small text.
- Transparent labels without white ink where readability matters.
- Finish regions too close to die-cut edges.
- Compressing a full wrap label onto the front of a cylindrical bottle.

## 软包装 / Flexible Packaging

柔性包装的工艺逻辑与硬盒完全不同：不能做击凸/压凹/厚重UV，核心层次感来自 **哑亮对比 + 烫金烫银 + 镭射膜 + 功能附件（拉链/吸嘴/易撕口）**。热封边（5-10mm）是工艺禁区。

---

### 自立袋 Stand-up Pouch (Doypack)
最常见软包装袋型，底部折叠可站立，通常顶部带拉链或吸嘴。

Recommend:
- 结构：底部折叠面（bottom gusset）+ 两侧热封边 + 顶部拉链/吸嘴。
- 材质：PET/AL/PE（高阻隔）、PET/VMPET/PE（镀铝）、MATT OPP/PE（哑面高级）、牛皮纸/PE（自然环保）。
- 工艺：大面积哑油 + Logo/产品名局部亮油对比 + 品牌标识烫金/烫银。拉链颜色与袋身设计统一。可加挂孔（Euro slot）。
- 适合：咖啡豆、茶叶、坚果零食、宠物食品、保健品、日化补充装。
- 表现要点：效果图需展示袋子装满内容物后的自然膨胀形态，底部折叠面微微展开，袋身有自然柔和的褶皱而非硬板平面。

Avoid:
- 硬盒工艺（击凸/压凹/厚重UV）。
- 烫金/丝印覆盖热封边区域。
- 空袋状态渲染（扁平无立体感）。
- 袋子像纸盒一样棱角分明（正确应为柔和的曲面转折）。

### 三边封袋 3-Side Seal Pouch
三边热封，扁平袋型，结构最简单，成本最低。

Recommend:
- 结构：左右+底部三边热封，顶部开口或带拉链/易撕口。
- 材质：PET/PE、OPP/CPP（透明）、PET/AL/PE、镀铝复合。
- 工艺：正面哑油/亮油对比 + 品牌Logo烫金 + 顶部易撕口（V型缺口或激光线）。
- 适合：一次性食品小包、调料包、面膜袋、试用装、小零食。
- 表现要点：扁平袋型，内容物少时微微鼓起。热封边清晰可见（锯齿边或直线边）。易撕口位置醒目。

Avoid:
- 自立袋的底部折叠结构（三边封袋没有底部折叠）。
- 内容物过满导致袋子鼓得像气球。

### 八边封袋 Quad-seal / Box Pouch
平底+两侧风琴折叠，可稳定站立，货架陈列效果好，比自立袋更有立体感。

Recommend:
- 结构：平底 + 左右两侧风琴（side gusset）+ 顶部拉链。展开后类似长方体。
- 材质：PET/AL/PE、PET/VMPET/PE、MATT OPP/PE。
- 工艺：正面哑油 + 图案局部亮油对比 + Logo烫金/烫银 + 侧面风琴区可做连续图案或纯色。顶部拉链 + 易撕口 + 挂孔。
- 适合：高端咖啡、精品茶叶、坚果大包装、宠粮大包装。
- 表现要点：装满后四面分明、风琴面自然展开、底部平坦、顶部拉链微微张开。袋身有柔和的薄膜褶皱痕迹。

Avoid:
- 八边封袋做成普通自立袋（忘记风琴结构）。
- 风琴面图案拉伸变形或断开。

### 拉链袋 Zipper Bag
顶部拉链封口，可重复开合，注重使用便利性。

Recommend:
- 结构：自立袋或三边封袋 + 顶部凹凸扣拉链或滑块拉链。
- 材质：PET/PE、MATT OPP/PE、镀铝复合。
- 工艺：袋身简约印刷 + 品牌Logo烫金 + 拉链区域明显标识（"按压密封"或滑块提示）。易撕口在拉链上方。
- 适合：零食、干果、冷冻食品、日化补充装、湿巾。
- 表现要点：拉链区域微微凸起，拉链颜色通常与袋身配色协调（白/黑/银/透明）。

Avoid:
- 拉链颜色与袋身设计冲突（如大红配大绿）。
- 拉链看起来像印刷的线而非实体塑料件。

### 吸嘴袋 Spout Pouch
自立袋+顶部焊接吸嘴+旋盖，液体/膏体专用。

Recommend:
- 结构：自立袋 + 顶部居中或偏侧吸嘴 + PP 旋盖或翻盖。
- 材质：PET/AL/PE（高阻隔）、PET/PE（透明内容物可见）。
- 工艺：袋身哑油为主 + Logo/品牌烫金或亮油对比 + 吸嘴和盖子颜色配合袋身色系。
- 适合：果汁、酸奶、果泥、洗衣液补充装、护肤品补充装。
- 表现要点：吸嘴与袋身的焊接过渡自然（热封焊接环），盖子比例恰当。袋子因液体内容物饱满圆润。

Avoid:
- 吸嘴比例过大或过小。
- 吸嘴看起来像浮在袋子上而非焊接上去。
- 固体食品袋用吸嘴（不物理）。

### 纸巾软包装 Tissue Soft Pack
PE 膜枕式包装，柔软、半透或印刷膜，低成本高销量。

Recommend:
- 结构：PE 膜包裹纸巾后三边热封成枕形，顶部/侧面有易撕口或撕拉线。
- 材质：纯 PE 膜（透明/乳白/印刷膜），单层或共挤。
- 工艺：柔印或凹印简单图案（品牌Logo+产品名+规格+卖点），印刷面积克制。易撕口/撕拉线是核心功能工艺。不做烫金/UV等精细工艺（成本不匹配且 PE 附着力差）。
- 适合：抽纸、手帕纸、湿巾外包装、厨房纸巾。
- 表现要点：包装自然柔软，有自然褶皱和内容物填充的饱满感。易撕口位置明显。膜面有柔和的自然反光弧面。

Avoid:
- 烫金/UV等不匹配纸巾成本结构的工艺。
- 包装像硬盒一样棱角分明。
- PE 膜过度平滑像玻璃纸（应有柔和的微皱）。

### 卫生用品枕式包装 Hygiene Pillow Pack
PE 膜枕式或三边封包装，卫生巾/尿不湿/护垫等。

Recommend:
- 结构：PE 复合膜三边封或枕式包裹，通常带易撕口或撕拉线。
- 材质：纯 PE 膜、PET/PE 复合（如需更好的印刷效果）。
- 工艺：柔印/凹印品牌色+产品信息+规格。易撕口是必须项。挂孔（部分产品用于挂钩陈列）。哑面处理为主。
- 适合：卫生巾、护垫、尿不湿、成人纸尿裤。
- 表现要点：柔软鼓胀的枕形，易撕口醒目，品牌色准确。包装有自然柔和的弯折感。

Avoid:
- 过于刚硬的盒型质感。
- 忘记易撕口。
- 过度设计工艺（品类以功能性和品牌色识别为主）。

## 软包装主动询问逻辑

当识别到输入为软包装时，按以下优先级确认：

1. **袋型**（自立袋/三边封/八边封/拉链袋/吸嘴袋/纸巾枕式/卫生用品枕式？）
2. **复合膜材质**（铝箔/镀铝/PET+PE/牛皮纸/纯PE？）
3. **功能附件**（拉链/吸嘴/易撕口/挂孔/开窗？）
4. **印刷工艺**（哑亮对比/烫金/局部UV/镭射？）

示例询问模板：
```text
检测到软包装袋型。在推荐工艺前需要确认几个关键点：
1. 袋型？A.自立袋 B.三边封袋 C.八边封袋 D.拉链袋 E.吸嘴袋 F.纸巾枕式包 G.不确定
2. 复合膜材质？A.铝箔复合(高阻隔) B.镀铝复合 C.PET/PE D.牛皮纸/PE E.纯PE F.你推荐
3. 需要功能附件吗？A.拉链 B.吸嘴+盖 C.易撕口 D.挂孔 E.开窗 F.不需要 G.你推荐
```

## Default Middle/High-End Recommendation

Use when the user says “不确定，你推荐”:
- 定位：中高端品牌包装。
- 材质：根据结构选白卡纸、灰板裱特种纸或 PET 不干胶。
- 表面：哑膜、触感膜或细腻纸感。
- Logo：局部 UV、烫银或轻微击凸。
- 产品名：轻微局部 UV 或击凸。
- 辅助图形：低对比专色或细腻纹理。
- 背景：极轻微纸张/膜材肌理。
- 风格：干净、克制、高级，不堆工艺。

## 瓶装包装 / Bottle Packaging

瓶装包装需同时考虑两个维度：**瓶身工艺** + **标签方案**。两者需协调一致，不能只给标签建议忽略瓶身。

### 玻璃瓶 + 不干胶标签（饮品/调味品/化妆品精华液）

Recommend:
- 瓶身：高白料/晶白料玻璃，可选蒙砂效果或局部喷涂（底部渐变）。
- 标签：银龙PET / 白PET / 珠光PET 不干胶，或铜版纸不干胶（暖调/自然系品牌）。
- 瓶身工艺克制：玻璃本体质感已是卖点，瓶身最多做蒙砂或底部小面积喷涂，标签区净瓶。
- 标签工艺：Logo 烫银/烫金 + 产品名局部 UV + 背景哑面。
- 适合风格：干净透亮、果汁/茶饮/高端调味品/精华液。

Avoid:
- 大面积喷涂遮盖玻璃通透感。
- 标签材质与瓶身质感冲突（如粗纤维特种纸配高透玻璃）。
- 透明标签贴在透明玻璃上导致文字不可读（需白墨底）。

### PET 瓶 + 收缩套标（饮料/日化/洗护）

Recommend:
- 瓶身：PET 透明/半透瓶，可选瓶身底部渐变喷涂增色。
- 标签：PETG 收缩套标，360° 包裹，可满版印刷 + 局部光油/哑油对比。
- 盖/泵头：配合瓶身色系，可选电镀银/金/玫瑰金泵头提升质感。
- 适合风格：快消饮品、沐浴露、洗发水、清洁剂。

Avoid:
- 收缩套标在异型瓶上收缩不均起皱。
- 瓶口处套标堆积过厚。
- 套标覆盖区与瓶身喷涂重叠导致视觉混乱。

### 亚克力瓶 + 丝印/烫金（高端护肤/香水/精华）

Recommend:
- 瓶身：亚克力厚壁注塑，高透或渐变色喷涂（底部浓→顶部淡），或全电镀镜面金/银。
- 标签：直接丝印（极简高端，无标签触感），或用透明 PET 标签 + 烫金 Logo。
- 瓶身工艺为主角，标签为辅。丝印位置通常在瓶身正面或背面中线。
- 盖：电镀金/银/玫瑰金，或亚克力厚盖+内嵌 Logo。
- 适合风格：高端护肤精华、香水、贵妇级化妆品。

Avoid:
- 丝印套色不准（多色丝印定位极难）。
- 电镀瓶身+大面积不干胶标签（附着力差，易起翘）。
- 亚克力瓶+热收缩套标（高温收缩可能损伤瓶身）。

### HDPE/PP 瓶 + 模内标（日化/沐浴露/洗发水）

Recommend:
- 瓶身：HDPE/PP 注塑或吹塑，可选白色/本色/着色。
- 标签：IML 模内标（标签与瓶身一体成型，永久不脱落），或 PP 不干胶标签。
- 瓶身表面通常为模具直接成型（亚光/细纹），不另做喷涂。
- 盖：翻盖/泵头，配色统一。
- 适合风格：日化大众款、家庭装洗护。

Avoid:
- 模内标瓶身上再做不干胶标签（画蛇添足）。
- HDPE 瓶身喷涂（表面能低，附着力差，不经济）。
- 过于复杂的多工艺（与大众日化定位不符）。

### 铝瓶 + 喷涂+丝印（高端喷雾/香水/功能饮料）

Recommend:
- 瓶身：铝材质，可全喷涂哑光/亮光色，或保留金属本色+局部丝印。
- 瓶身工艺：喷涂打底 + 丝印品牌Logo/文字，或冲压 Logo + 喷涂填色。
- 标签：通常无标签（丝印直接上瓶身），或小尺寸不干胶贴于背面/底部。
- 盖：铝盖/PP盖，配合瓶身色系，可选电镀。
- 适合风格：高端喷雾、功能饮料、运动水壶、香水。

Avoid:
- 大面积不干胶标签遮盖铝瓶喷涂效果。
- 铝瓶表面不平整导致丝印边缘模糊。
- 喷涂前未做金属底漆导致掉漆。

### 陶瓷瓶 + 烤花（白酒/工艺品/香薰）

Recommend:
- 瓶身：陶瓷高温烧制，白釉/青釉/哑釉/窑变釉可选。
- 瓶身工艺：烤花（贴花纸+高温烧结，图案永久融入釉面），或手工描绘（高端定制）。
- 标签：通常无标签（烤花即标识），或颈部挂吊牌+底部贴标。
- 盖：木塞/陶瓷盖/金属旋盖，传统风格。
- 适合风格：高端白酒、黄酒、香薰瓶、工艺收藏品。

Avoid:
- 不干胶标签贴陶瓷（破坏传统质感）。
- 收缩套标覆盖陶瓷瓶（完全浪费陶瓷材质感）。
- 烤花图案过于复杂导致烧结起泡/偏色。

## 瓶装包装主动询问逻辑

当识别到输入为瓶装包装时，按以下优先级向用户确认：

1. **瓶身材质**（玻璃/PET/亚克力/PP-PE/铝/陶瓷？）→ 不确认则不继续推荐
2. **瓶身工艺**（喷涂/电镀/蒙砂/丝印/注塑本色？）→ 若用户说「你推荐」，根据品类给默认方案
3. **标签方式**（不干胶/收缩套标/模内标/直接丝印？）→ 结合瓶身材质推荐兼容方式
4. **标签材质工艺**（PET/纸类/烫金/UV？）→ 使用现有标签工艺速查表

示例询问模板：
```text
检测到瓶装包装。在推荐工艺前需要确认几个关键点：
1. 瓶身材质？A.玻璃 B.PET C.亚克力 D.PP/PE E.铝 F.陶瓷 G.不确定
2. 瓶身需要表面工艺吗？A.不需要(保留材质本色) B.喷涂 C.电镀 D.蒙砂 E.丝印 F.你推荐
3. 标签方式？A.不干胶标签 B.收缩套标 C.直接丝印(无标签感) D.你推荐
```
