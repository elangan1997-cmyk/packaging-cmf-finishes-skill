# Structure Recommendations

Use this reference when recommending materials and finishes by packaging structure.

## 天地盖 / 抽屉盒 / 礼盒

Recommend:
- 灰板裱特种纸、铜版纸裱灰板、艺术纸裱灰板。
- 哑膜、触感膜、不覆膜纸感方案。
- Logo 烫金/烫银/击凸；产品名局部 UV；背景轻压纹。
- Suitable for high-end gifts, brand sets, cosmetics, tea, wine, supplements, boutique products.

Avoid:
- Overfilled foil/UV.
- Large glossy lamination if the brand wants premium subtlety.
- Processes too close to folds, lid edges, or box corners.

## 插舌盒 / 扣底盒 / 常规彩盒

Recommend:
- 白卡纸、单铜纸、白卡覆哑膜/亮膜。
- Logo 局部 UV; small foil accents for premium lines; feature badge spot UV.
- Cost-aware retail/e-commerce finish plans.

Avoid:
- Heavy gift-box processes that mismatch low-cost structures.
- Thick UV or emboss over crease/tuck areas.
- Dense small text with foil/UV.

## 飞机盒 / 瓦楞盒 / 电商运输盒

Recommend:
- 瓦楞纸、坑纸、牛皮纸、白卡裱坑。
- Single-color or spot-color printing; bold brand marks; simple line graphics.
- Minimal finish; use structure and material texture as the premium point.

Avoid:
- Fine gradients and tiny copy on rough corrugated material.
- Complex foil/UV over compression folds.
- Making a shipping box look like a rigid luxury gift box unless requested.

## 瓶贴 / 标签 / 套标

Recommend:
- 银龙 PET、哑银 PET、白 PET、透明 PET、铜版纸不干胶。
- Spot UV, raised UV, white ink underprint, foil, reverse UV, tactile varnish.
- For transparent PET, mention white ink underprint for readability.
- For metallic PET, control contrast; do not make the whole label mirror chrome unless requested.

Avoid:
- Too-thin small text.
- Transparent labels without white ink where readability matters.
- Finish regions too close to die-cut edges.
- Compressing a full wrap label onto the front of a cylindrical bottle.

## Soft Pouches

Recommend:
- Matte film, aluminum composite film, pearlescent film, spot gloss contrast.
- Large main visual with controlled spot UV or matte/gloss contrast.

Avoid:
- Rigid emboss effects that ignore flexible material.
- Heavy foil on flexible fold/crimp zones unless production supports it.

## Default Middle/High-End Recommendation

Use when the user says “不确定，你推荐”:
- 定位：中高端品牌包装。
- 材质：根据结构选白卡纸、灰板裱特种纸或 PET 不干胶。
- 表面：哑膜、触感膜或细腻纸感。
- Logo：局部 UV、烫银或轻微击凸。
- 产品名：轻微局部 UV 或击凸。
- 辅助图形：低对比专色或细腻纹理。
- 背景：极轻微纸张/膜材肌理。
- 风格：干净、克制、高级，不堆工艺。
